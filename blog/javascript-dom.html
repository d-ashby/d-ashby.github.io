JavaScript fundamentals

JavaScript and its relationship to HTML and CSS

JavaScript acts as a programming 'layer' to the elements rendered by the browsers interpretation of HTML and CSS. HTML represents the elements of a page where css applies style to those elements and is then rendered as what is known as the Document Object Model (DOM for short). Javascript can further manipulate this DOM to add elements, styles, and other more complex interactions.

Control flow and loops, arrays, and objects

Control Flow relates to how javascript is read by the browser. It runs in a sequential order from top to bottom i.e. first line to the last line of code.
Loops are used within javascript to do a certain task repeatedly.
